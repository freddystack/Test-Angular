Esta es un repaso de como se creo esta aplicación y cual es su función

Estos son los componentes usados

  • Componente Aside
  • Componente Jobs
  • Componente People
  • Componente About

como esta consumiendo las apis esta aplicación?

Estamos utilizando el httpClient para consumir la información de las apis de Jobs y Peoples

Otras tecnologias utilizadas

  • Angular CDk (Virtual Scroll)
  • Animate CSS
  • Bootstrap
  • Angular Rounting

Qué es capaz de hacer esta aplicación?

Esta aplicación funciona consumiendo recursos de apis externas de esta manera podemos utilizar su información. En esta aplicación podemos acceder a la lista de trabajos que se muestra en la pestaña de Jobs, la lista de personas que se muestra en la pestaña de Peoples ademas de poder ver a datalle su información y finalmente tenemos la sección de About que nos muestra una pequeña idea de como se creo esta aplicación
